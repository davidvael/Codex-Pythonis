Codex Pythonis is an independently authored technical manuscript dedicated to the structure, principles, and applied understanding of the Python programming language. Formed through direct study and experimentation, it exists as a living reference for disciplined learning and open technical knowledge.

---

## Overview

Codex Pythonis is maintained as a continuous manuscript and educational resource.  
It is designed with clarity, technical precision, and long-term maintainability in mind.

This repository serves as the official public archive of the manuscript.
